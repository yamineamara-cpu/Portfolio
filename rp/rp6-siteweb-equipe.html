Contexte

Dans le cadre d'un projet de formation en développement web collaboratif, j'ai participé à la réalisation d'un site web en équipe, avec une organisation de projet gérée via Trello.

L'équipe a suivi une méthodologie de type Kanban pour répartir les tâches, suivre l'avancement et respecter les délais. Chaque membre s'est vu attribuer des responsabilités spécifiques (front-end, back-end, intégration, tests).

Ce projet a permis de mettre en pratique à la fois les compétences techniques (développement web) et les compétences organisationnelles (gestion de projet, travail en équipe, communication).

🌐 HTML5 / CSS3 🐘 PHP 🗄️ MySQL 📋 Trello (Kanban) 💻 VS Code 👥 Travail en équipe
