Contexte

Dans le cadre des cours de bases de données du BTS SIO, j'ai étudié et analysé les différentes solutions de stockage de données disponibles, ainsi que les méthodes de modélisation des données.

Ce travail d'introduction aux bases de données couvre les concepts fondamentaux : les modèles de données (MCD, MLD, MPD), les systèmes de gestion de bases de données relationnelles (SGBDR) et les différentes solutions de stockage (fichiers plats, BDD relationnelles, NoSQL).

Le travail a été formalisé dans un document structuré présentant les analyses et comparaisons effectuées.
