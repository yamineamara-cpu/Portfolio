Contexte

Dans le cadre de l'épreuve E5 du BTS SIO, j'ai conçu et développé ce portfolio numérique entièrement en HTML / CSS sans framework, afin de présenter mes réalisations professionnelles, ma veille technologique et les documents destinés au jury.

La conception a nécessité une réflexion sur l'architecture de l'information, l'ergonomie, le design responsive et l'accessibilité — conformément aux standards du web professionnel. Le portfolio est structuré en 4 sections : Curriculum, Réalisations, Veille et Rectorat.

Plusieurs versions ont été réalisées afin d'explorer différentes approches de design : version classique, version minimaliste et version avec effets parallax.

🌐 HTML5 🎨 CSS3 ⚡ JavaScript 📱 Responsive Design 💻 VS Code 📐 CSS Grid / Flexbox 🔤 Google Fonts
