Contexte

Dans le cadre des travaux pratiques de SQL du BTS SIO, j'ai appris à exploiter des données stockées dans une base de données relationnelle à l'aide du langage normalisé SQL.

Ce TP porte sur les requêtes SELECT monotable : sélection de colonnes, filtrage avec WHERE, tri avec ORDER BY, et fonctions d'agrégation (COUNT, SUM, AVG).

L'objectif est de maîtriser l'interrogation d'une base de données afin d'en extraire des informations pertinentes pour répondre à des besoins métier.
